Sobre o Code Queens
Um projeto para aprender programação com leveza, identidade e coragem para começar.
✦ Sobre o Code Queens

O Code Queens é um projeto e blog criado para aprender programação de um jeito leve, simples e acolhedor.

Aqui, conceitos que parecem difíceis ganham uma linguagem mais próxima, exemplos práticos e uma estética própria.

"Programação não precisa parecer impossível para ser poderosa."

O foco é ajudar quem está começando ou quer fortalecer a base, sem medo de errar e sem complicar o que pode ser explicado com clareza.

✦ Quem é sheyla_ghost

sheyla_ghost é a criadora do Code Queens: uma estudante e dev em evolução, apaixonada por tecnologia, criatividade, comunidade e aprendizado.

O projeto nasce dessa vontade de transformar estudo em jornada, juntando código, identidade visual e uma forma mais humana de explicar programação.

"Aprender também é construir uma versão nova de si mesma."

Mais do que mostrar respostas prontas, o Code Queens quer acompanhar o processo: dúvidas, tentativas, erros, descobertas e pequenos avanços que viram confiança.

✦ Missão do projeto
Programação para começar Leve · Acessível · Divertida Tornar o código menos assustador e mais possível.
Para quem é

Meninas, mulheres e qualquer pessoa que queira começar na tecnologia com mais confiança.

Como a gente aprende

Com linguagem simples, exemplos claros, estética própria e espaço para evoluir no próprio ritmo.

O objetivo

Mostrar que código também pode ser criativo, acessível e feito por nós.
